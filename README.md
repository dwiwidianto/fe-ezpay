## Installation


#### With Docker

install [docker-compose](https://docs.docker.com/compose/install/) & running 

```bash
docker-compose up -d
```
stop service
```bash
docker-compose down
```

#### Running in local computer
```bash
yarn install
```
```bash
yarn start
```
